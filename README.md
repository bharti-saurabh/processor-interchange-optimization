# Processor Interchange Optimisation

**Segment:** Processor | **Category:** Benchmarking | **Owner:** Straive Strategic Analytics | **Year:** 2024

## Objective
Identify interchange downgrade patterns costing the processor's merchants basis points of margin, and simulate the GMV impact of qualification improvements.

## Methodology
1. Interchange waterfall — map each transaction to actual vs optimal rate tier
2. Downgrade root-cause: missing L2/L3 data, late settlement, missing AVS, no 3DS
3. Network fee benchmarking vs published rates and peer processors
4. Routing optimisation: least-cost routing for dual-network debit
5. Intervention ROI sizing

## Key Findings
| Issue | Downgrade Rate | Estimated Annual Cost |
|---|---|---|
| Missing Level 2 Data | 12.4% of B2B txns | $2.1M |
| Late Settlement (>2 days) | 3.8% | $0.8M |
| CNP without 3DS | 6.2% | $1.4M |
| Total Opportunity | — | ~$4.3M |
